HScroll.js

Introduction

A simple, dependency free javascript library that enables vertically scrolling an overflowed element by hovering over on its edges.

The scrolling speed is mapped to a gradient, speeding up close to the edges and tapering off to zero in the middle. For a use case checkout Apples excellent 30 years website

How to use it

hScroll = new HScroll(".scroll");
hScroll.destroy();
